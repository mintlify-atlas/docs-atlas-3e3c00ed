XNNPACK is a highly optimized library of neural network inference operators for mobile, server, and Web platforms. It provides low-level performance primitives for accelerating high-level machine learning frameworks across ARM64, ARMv7, x86/x86-64, WebAssembly, RISC-V, and Hexagon architectures.

## Why XNNPACK?

XNNPACK delivers state-of-the-art inference performance through hand-tuned SIMD microkernels, efficient memory layouts, and an optimizing subgraph runtime.

<CardGroup cols={3}>
  <Card title="Multi-platform SIMD" icon="microchip">
    Optimized kernels for NEON, AVX512, WebAssembly SIMD, and Hexagon HVX.
  </Card>
  <Card title="Multiple data types" icon="binary">
    FP32, FP16, BF16, INT8 (QS8/QU8), and 4-bit quantized inference.
  </Card>
  <Card title="Multi-threaded" icon="layer-group">
    Parallel inference via pthreadpool across all big cores.
  </Card>
</CardGroup>
